# Konjam (A Little Tamil)

Welcome to the Konjam project. This project is dedicated to helping expats, tech workers, and newcomers learn street-level, practical survival Tamil in Chennai.

## Overview
Konjam meaning "a little" in Tamil, focuses on actionable phrases, phonetics, and interactive exercises to get you communicating on the streets of Chennai effectively.

## Mission
To learn the essentials of spoken Tamil to get things done—whether negotiating with an auto driver, talking to shopkeepers, or getting around.
